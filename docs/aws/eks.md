# Elastic Kubernetes Service (GUI)
Requirements
- [] IAM [Cluster Role for EKS](https://docs.aws.amazon.com/eks/latest/userguide/service_IAM_role.html#create-service-role)
- [] Configure Cluster in EKS page


## EKS IAM Role
## Trusted Entity 
`{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Effect": "Allow",
            "Principal": {
                "Service": [
                    "eks.amazonaws.com"
                ]
            },
            "Action": "sts:AssumeRole"
        }
    ]
}`
